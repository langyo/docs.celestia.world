# Записи Архитектурных Решений (ADR)

В этой директории записаны ключевые архитектурные решения, принятые в ходе разработки Entelecheia. Каждый ADR объясняет **что** было решено, **почему** это было решено и какие **компромиссы** были рассмотрены.

ADR следуют [шаблону ADR Майкла Найгарда](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). Они неизменны после публикации — заменённые решения помечаются соответствующим образом.
