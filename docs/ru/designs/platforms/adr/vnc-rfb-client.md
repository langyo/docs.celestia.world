# Клиент протокола VNC (RFB)

- **Статус**: Принято
- **Дата**: 2026-06-09
- **Авторы**: Evernight Core Team

## Контекст

Evernight требуется графический клиент удалённого рабочего стола с поддержкой VNC. Протокол RFB (RFC 6143) является стандартом для VNC. Варианты: использовать существующий крейт Rust, привязаться к libvncclient или реализовать с нуля.

## Решение

Реализовать клиент RFB 003.008 на чистом Rust с нуля. Поддерживает рукопожатие версий, согласование безопасности (None, VncAuth), аутентификацию с запросом-ответом на основе DES, согласование формата пикселей и кодирование Raw/CopyRect. Реализует типаж `ViewportBackend` для интеграции с фронтендом.

## Последствия

### Положительные

- Отсутствие C-зависимостей; полный контроль над потоком протокола.

### Отрицательные

- Кодирование ZRLE и Tight пока не реализовано (сложны в реализации).
- Потребовалась реализация DES для аутентификации VNC.
