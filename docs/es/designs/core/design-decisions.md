# Registros de Decisiones de Arquitectura (ADR)

Este directorio registra las decisiones arquitectónicas clave tomadas durante el desarrollo de Entelecheia. Cada ADR explica **qué** se decidió, **por qué** se decidió y qué **compromisos** se consideraron.

Los ADR siguen la [plantilla ADR de Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions). Son inmutables una vez publicados — las decisiones reemplazadas se marcan en consecuencia.
