# 架构决策记录（ADR）

本目录记录了 Entelecheia 开发过程中做出的关键架构决策。每个 ADR 解释了**决定了什么**、**为什么**决定以及考虑了哪些**权衡**。

ADR 遵循 [Michael Nygard ADR 模板](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions)。一旦发布即不可变——被取代的决策会相应标记。
