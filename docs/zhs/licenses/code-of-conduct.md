# 行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺使每位参与者在我们的社区中都能获得免受骚扰的体验，无论其年龄、体型、可见或不可见的残疾、民族、性征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教信仰，还是性身份与性取向如何。

我们承诺以有助于营造开放、热情、多元、包容且健康社区的方式行事和互动。

## 我们的标准

有助于为我们社区营造积极环境的行为示例包括：

* 对他人表现出同理心和善意
* 尊重不同的意见、观点和经验
* 给予并从容接受建设性反馈
* 为我们的错误承担责任，向受影响者道歉，并从中吸取经验
* 关注的不仅是我们个人的利益，更是整个社区的共同利益

不可接受的行为示例包括：

* 使用带有性意味的语言或图像，以及任何形式的性关注或性挑逗
* 恶意挑衅、侮辱性或贬损性评论，以及针对个人或政治的攻击
* 公开或私下骚扰
* 未经他人明确许可，发布其私人信息，例如物理地址或电子邮件地址
* 在专业场合下可被合理视为不当的其他行为

## 执行责任

社区领导者有责任澄清和执行我们可接受行为的准则，并将对他们认为不当、具有威胁性、冒犯性或有害的任何行为采取适当且公正的纠正措施。

社区领导者有权且有责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、维基编辑、议题和其他贡献，并在适当时说明做出审核决定的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间正式代表社区的情况。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账号发帖，或在在线或线下活动中担任被任命的代表。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可向负责执行的社区领导者报告，联系方式：contact@celestia.world。所有投诉都将得到及时且公正的审查和调查。

所有社区领导者都有义务尊重任何事件报告者的隐私和安全。

## 执行指南

社区领导者在判定任何行为违反本行为准则的后果时，将遵循以下"社区影响指南"：

### 1. 纠正

**社区影响**：使用不当语言，或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：由社区领导者发出私下的书面警告，澄清违规的性质并解释该行为为何不当。可能会被要求作出公开道歉。

### 2. 警告

**社区影响**：通过单次事件或一系列行为造成的违规。

**后果**：发出警告，并对持续的行为设定后果。在规定期限内，不得与相关人员互动，包括主动联系执行行为准则的人员。这包括在社区空间以及社交媒体等外部渠道中避免互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在规定期限内，临时禁止与社区进行任何形式的互动或公开交流。在此期间，不允许与相关人员有任何公开或私下互动，包括主动联系执行行为准则的人员。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的惯常模式，包括持续的不当行为、对个人的骚扰，或对某类群体的攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
