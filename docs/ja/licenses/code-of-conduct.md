# 行動規範

## 私たちの誓い

私たちメンバー、貢献者、およびリーダーは、年齢、体格、目に見えるまたは見えない障がい、民族、性的特徴、性自認と表現、経験の程度、教育、社会経済的地位、国籍、外見、人種、宗教、または性的アイデンティティと指向にかかわらず、コミュニティへの参加がすべての人にとってハラスメントのない体験となるよう誓います。

私たちは、開かれ、歓迎的で、多様性と包摂性を備え、健全なコミュニティに貢献するような方法で行動し、交流することを誓います。

## 私たちの基準

私たちのコミュニティにおいて前向きな環境づくりに貢献する行動の例：

* 他者への共感と思いやりを示すこと
* 異なる意見、視点、経験を尊重すること
* 建設的なフィードバックを与え、また穏やかに受け入れること
* 自身の過ちに対して責任を負い、影響を受けた人に謝罪し、その経験から学ぶこと
* 個人としてだけでなく、コミュニティ全体にとって最善であることに注力すること

許容されない行動の例：

* 性的な言語や画像の使用、およびあらゆる形態の性的な注目や言い寄り
* 荒らし、侮辱的または中傷的なコメント、および個人的または政治的な攻撃
* 公的な、または私的なハラスメント
* 物理的住所やメールアドレスなど、他者の個人情報を明示的な許可なく公開すること
* 専門的な場において不適切と合理的にみなされうるその他の行動

## 執行の責任

コミュニティリーダーは、許容される行動の基準を明確にし執行する責任を負い、不適切、脅威的、攻撃的、または有害とみなすいかなる行動に対しても、適切かつ公正な是正措置を講じます。

コミュニティリーダーは、本行動規範に沿わないコメント、コミット、コード、ウィキ編集、イシュー、その他の貢献を削除、編集、または拒否する権利と責任を有し、適切な場合にはそのモデレーション決定の理由を伝えます。

## 適用範囲

本行動規範は、すべてのコミュニティ空間に適用されるとともに、個人が公共の場でコミュニティを公式に代表する場合にも適用されます。コミュニティを代表する例には、公式メールアドレスの使用、公式ソーシャルメディアアカウントによる投稿、オンラインまたはオフラインのイベントで任命された代表者として振る舞うことなどがあります。

## 執行

虐待的、ハラスメント、またはその他の許容されない行動の事例は、執行を担当するコミュニティリーダーまで contact@celestia.world までご報告ください。すべての通報は迅速かつ公正に審査および調査されます。

すべてのコミュニティリーダーは、いかなる事象の報告者のプライバシーと安全を尊重する義務を負います。

## 執行ガイドライン

コミュニティリーダーは、本行動規範への違反とみなす行動の結果を決定するにあたり、以下の「コミュニティ影響ガイドライン」に従います。

### 1. 是正

**コミュニティへの影響**：不適切な言語の使用、またはコミュニティにおいて非専門的もしくは歓迎されないとみなされるその他の行動。

**結果**：コミュニティリーダーからの私的な文面による警告。違反の性質を明確にし、当該行動がなぜ不適切であったかを説明します。公開での謝罪が求められる場合があります。

### 2. 警告

**コミュニティへの影響**：単一の事象、または一連の行動による違反。

**結果**：以降の行動に対する結果を伴う警告。関与した人々との、行動規範を執行する人々への一方的な連絡を含め、一定期間の交流を禁じます。これには、コミュニティ空間での交流のほか、ソーシャルメディアのような外部チャンネルでの交流の回避が含まれます。これらの条件に違反した場合、一時的または恒久的な追放につながる可能性があります。

### 3. 一時的追放

**コミュニティへの影響**：継続的な不適切行動を含む、コミュニティ基準の重大な違反。

**結果**：一定期間、コミュニティとのあらゆる種類の交流や公的なコミュニケーションを一時的に禁止します。この期間中、行動規範を執行する人々への一方的な連絡を含め、関与した人々との公的または私的な交流は一切許可されません。これらの条件に違反した場合、恒久的な追放につながる可能性があります。

### 4. 恒久的追放

**コミュニティへの影響**：継続的な不適切行動、個人へのハラスメント、または特定の集団に対する攻撃や侮蔑を含む、コミュニティ基準違反の常習的なパターンの示唆。

**結果**：コミュニティ内でのあらゆる種類の公的交流を恒久的に禁止します。
