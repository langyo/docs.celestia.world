# 行為準則

## 我們的承諾

作為成員、貢獻者和領導者，我們承諾使每位參與者在我們的社群中都能獲得免受騷擾的體驗，無論其年齡、體型、可見或不可見的障礙、民族、性徵、性別認同與表達、經驗水平、教育背景、社經地位、國籍、個人外貌、種族、宗教信仰，還是性身份與性取向如何。

我們承諾以有助於營造開放、熱情、多元、包容且健康社群的方式行事和互動。

## 我們的標準

有助於為我們社群營造積極環境的行為示例包括：

* 對他人表現出同理心和善意
* 尊重不同的意見、觀點和經驗
* 給予並從容接受建設性回饋
* 為我們的錯誤承擔責任，向受影響者道歉，並從中汲取經驗
* 關注的不僅是我們個人的利益，更是整個社群的共同利益

不可接受的行為示例包括：

* 使用帶有性意味的語言或圖像，以及任何形式的性關注或性挑逗
* 惡意挑釁、侮辱性或貶損性評論，以及針對個人或政治的攻擊
* 公開或私下騷擾
* 未經他人明確許可，發布其私人資訊，例如實體地址或電子郵件地址
* 在專業場合下可被合理視為不當的其他行為

## 執行責任

社群領導者有責任釐清和執行我們可接受行為的標準，並將對他們認為不當、具有威脅性、冒犯性或有害的任何行為採取適當且公正的糾正措施。

社群領導者有權且有責任刪除、編輯或拒絕與本行為準則不符的評論、提交、程式碼、維基編輯、議題和其他貢獻，並在適當時說明做出審核決定的原因。

## 適用範圍

本行為準則適用於所有社群空間，也適用於個人在公共空間正式代表社群的情況。代表我們社群的示例包括使用官方電子郵件地址、透過官方社群媒體帳號發文，或在線上或線下活動中擔任被任命的代表。

## 執行

如遇濫用、騷擾或其他不可接受的行為，可向負責執行的社群領導者報告，聯絡方式：contact@celestia.world。所有投訴都將得到及時且公正的審查和調查。

所有社群領導者都有義務尊重任何事件報告者的隱私和安全。

## 執行指南

社群領導者在判定任何行為違反本行為準則的後果時，將遵循以下「社群影響指南」：

### 1. 糾正

**社群影響**：使用不當語言，或其他在社群中被視為不專業或不受歡迎的行為。

**後果**：由社群領導者發出私下的書面警告，釐清違規的性質並解釋該行為為何不當。可能會被要求作出公開道歉。

### 2. 警告

**社群影響**：透過單次事件或一系列行為造成的違規。

**後果**：發出警告，並對持續的行為設定後果。在規定期限內，不得與相關人員互動，包括主動聯絡執行行為準則的人員。這包括在社群空間以及社群媒體等外部頻道中避免互動。違反這些條款可能導致暫時或永久封鎖。

### 3. 暫時封鎖

**社群影響**：嚴重違反社群標準，包括持續的不當行為。

**後果**：在規定期限內，暫時禁止與社群進行任何形式的互動或公開交流。在此期間，不允許與相關人員有任何公開或私下互動，包括主動聯絡執行行為準則的人員。違反這些條款可能導致永久封鎖。

### 4. 永久封鎖

**社群影響**：表現出違反社群標準的慣常模式，包括持續的不當行為、對個人的騷擾，或對某類群體的攻擊或貶低。

**後果**：永久禁止在社群內進行任何形式的公開互動。
