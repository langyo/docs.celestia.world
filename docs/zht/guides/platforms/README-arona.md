> Part of the [celestia-island](https://github.com/celestia-island) ecosystem.

JSON-RPC 2.0 協定型別、TypeScript 繫結與文件中樞。由 entelecheia 與 shittim-chest 所使用。

## 快速開始

```bash
# 建構
cargo build

# 執行所有測試（包含 TypeScript 繫結生成）
cargo test --all-features

# 檢查程式碼風格與格式
cargo clippy --all-targets --all-features -- -D warnings
cargo fmt --all -- --check

# 僅生成 TypeScript 繫結
cargo test --package arona
```

或使用 [just](https://github.com/casey/just) 任務執行器：

```bash
just build
just test
just fmt-check
```
